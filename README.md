# Hold Shift to Select Multiple Checkboxes

A mini practice project built as part of **project-based learning** to explore DOM manipulation, event handling, and keyboard interactions using **Vanilla JavaScript**.

This project recreates a common email client feature where holding the **Shift** key while selecting another checkbox automatically selects every checkbox between the two.

---

## 🎯 Purpose

The goal of this project is to gain hands-on experience with JavaScript by implementing a real-world UI interaction from scratch without using any external libraries or frameworks.

---

## 📚 Learning Outcomes

Through this project, I learned how to:

- Select and manipulate DOM elements
- Handle mouse and keyboard events together
- Detect modifier keys using `event.shiftKey`
- Track application state using JavaScript
- Iterate through DOM collections
- Implement range selection logic between two elements
- Build interactive UI behavior with Vanilla JavaScript

---

## 🚀 Project Outcome

Successfully implemented the **Shift + Click multi-selection** functionality commonly found in email clients, task managers, and productivity applications using only HTML, CSS, and JavaScript.
